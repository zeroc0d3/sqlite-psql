# sqlite-psql
Dump data from sqlite to postgreSQL

### Step
* Clone this repo
* Requirements :
  - Python 2.7.3
  - SQLite 3.7.9 
* Set your source sqlite & target folder in "run_dump.sh"
* Open your terminal, and run: sh run_dump.sh
* DONE.

### Export sqlite
* Export schema & data (all)
* Export schema (only)
* Export data (only)
  - Run: /bin/sh sqlite_export.sh
